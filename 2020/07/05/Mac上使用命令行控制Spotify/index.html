安装shpotify

用来控制Spotify的第三方脚本

使用homebrew安装：

brew install shpotify

Github地址



链接SpotifyAPI

进入Spotify开发者控制面板DashBoard

点击Log In登录Spotify账号



在登陆成功后的网页

点击CREATE AN APP来创建一个App



名字和描述随意写，下面两个选项打上勾✔️，然后点CREATE按钮



创建好后点击SHOW CLIENT SECRET按钮



将Client ID 和 Client Secret 保留

然后去终端cd到家目录下

cd ~

用本文编辑器打开一个名为.shpotify.cfg的配置文件

使用英文半角双引号将ID和Secret括起来，不要有多余的空格
